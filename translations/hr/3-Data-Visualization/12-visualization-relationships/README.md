## Facet gridovi

## Dvostruki linijski grafovi

Isprobajte vi≈°elinijski graf superponiranjem dvaju linijskih grafova jedan na drugi, koristeƒái Seabornov 'despine' za uklanjanje gornjih i desnih osi, te koristeƒái `ax.twinx` [iz Matplotliba](https://matplotlib.org/stable/api/_as_gen/matplotlib.axes.Axes.twinx.html). Twinx omoguƒáuje dijagramu dijeljenje x osi i prikaz dviju y osi. Dakle, prika≈æite prinos po koloniji i broj kolonija, superponirano:

```python

lineplot = sns.lineplot(x=honey['year'], y=honey['numcol'], data=honey, 
                        label = 'Number of bee colonies', legend=False)
sns.despine()
plt.ylabel('# colonies')
plt.title('Honey Production Year over Year');

ax2 = ax.twinx()
lineplot2 = sns.lineplot(x=honey['year'], y=honey['yieldpercol'], ax=ax2, color="r", 
                         label ='Yield per colony', legend=False) 
sns.despine(right=False)
plt.ylabel('colony yield')
ax.figure.legend();
```
